# J-J-Health-for-Humanity-ESG-Goals-Tracker
J&amp;J publishes 5 Health for Humanity goals across carbon, packaging, water, supplier sustainability — but no single view reconciles them against peer pharma. Approach: SQL schema ingesting CDP + SEC EDGAR + EPA disclosures; Python ETL; Chart.js dashboard with year filter, net-zero pathway overlay, and Merck/BMS/Pfizer benchmarking.
